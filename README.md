# 🌱 EcoAgro 360

## Agro Forte, Futuro Sustentável: equilíbrio entre produção e meio ambiente

O **EcoAgro 360** é um projeto desenvolvido para o **Agrinho 2026 – Categoria Desenvolvimento Front-End**, com o objetivo de demonstrar, por meio de uma experiência interativa, que é possível conciliar alta produtividade agrícola com responsabilidade ambiental.

O sistema coloca o usuário no papel de gestor de uma propriedade rural, permitindo que tome decisões estratégicas relacionadas ao uso de energia, irrigação, preservação ambiental, fertilizantes e manejo do solo. Cada escolha influencia diretamente indicadores como economia, biodiversidade, qualidade do solo, consumo de água e emissão de carbono.

---

# 🎯 Objetivos

- Demonstrar a importância da agricultura sustentável.
- Incentivar o uso consciente dos recursos naturais.
- Mostrar o impacto das decisões do produtor rural.
- Sensibilizar sobre o equilíbrio entre desenvolvimento econômico e preservação ambiental.
- Promover educação ambiental de forma dinâmica e interativa.

---

# 🚜 Tema do Agrinho 2026

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

O projeto foi desenvolvido para representar como tecnologia, inovação e boas práticas agrícolas podem contribuir para uma produção eficiente sem comprometer os recursos naturais das futuras gerações.

## Simulador

O jogador pode tomar decisões como:

- ☀️ Implantar energia solar;
- ⛽ Utilizar diesel;
- 🌿 Aplicar fertilizantes orgânicos;
- 🧪 Utilizar fertilizantes químicos;
- 💧 Irrigação inteligente;
- 🚿 Irrigação convencional;
- 🌾 Rotação de culturas;
- 🌽 Monocultura;
- 🌳 Preservar mata nativa;
- 🪓 Desmatar áreas.

Cada ação modifica indicadores ambientais e econômicos.

---

## Eventos Climáticos

Durante a simulação podem ocorrer eventos aleatórios:

- ☀️ Seca prolongada;
- 🌧️ Chuvas abundantes;
- 🐛 Pragas agrícolas;
- 🐝 Polinização elevada;
- 🌪️ Tempestades;
- 🌱 Recuperação natural do solo.

---

## Dashboard

Monitoramento em tempo real de:

- 🌱 Solo;

- 💰 Economia;
- 🌳 Biodiversidade;
- 🌍 Emissão de carbono.

---

## Relatório Final

Ao concluir a simulação, o sistema apresenta:

- Pontuação final;
- Classificação do produtor;
- Medalhas conquistadas;
- Recomendações personalizadas;
- Indicadores ambientais;
- Melhor pontuação salva localmente.

---

# 🏆 Sistema de Classificação

Dependendo do desempenho obtido:

| Pontuação | Classificação |
|------------|-----------------------------|
| até 199 | 🥉 Produtor em Desenvolvimento |
| 200 a 279 | 🥈 Agricultor Consciente |
| 280 a 349 | 🥇 Guardião da Natureza |
| 350 ou mais | 💎 Mestre do Agro Sustentável |
