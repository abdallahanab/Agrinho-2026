# EcoAgro 360 🌱

Projeto desenvolvido para o Agrinho 2026.

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo

Simular decisões agrícolas e mostrar como elas impactam o meio ambiente, a economia e a qualidade de vida.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- LocalStorage
- GitHub Pages

## Funcionalidades

- Simulador interativo
- Dashboard ambiental
- Eventos climáticos
- Sistema de pontuação
- Relatório final personalizado
- Responsivo para celular e computador
